### What's New in This Release

#### HIP API Change

The following was added to the HIP API in the ROCm 5.6.0 release:

##### `hipStreamGetDevice`

- Get the device associated with the stream.

    ```h
    hipError_t hipStreamGetDevice(hipStream_t stream, hipDevice_t* device);
    ```
